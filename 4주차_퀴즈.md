### 1. 아래 코드의 결과와 그 이유를 설명해주세요(선민)
``` java
class A {
  A() {
    System.out.println("A 클래스 인스턴스 생성");
  }
}

class B extends A {
  B() {
    System.out.println("B 클래스 인스턴스 생성");
  }

  B(int num) {
    System.out.println("B 클래스 인스턴스 " + num + " 생성");
  }
}

class C extends B {
  C() {
    super(1);
    System.out.println("C 클래스 인스턴스 생성");
  }
}

public class Main {
    public static void main(String[] args) {
        B b = new C();
    }
}
```
