##### 4. 아래 코드의 출력 결과 및 이유를 설명하세요.(선민)
```java
public class Main {
    public static void main(String[] args) {
        int num = Integer.MAX_VALUE;
        num = num + 1;
        
        System.out.println(num);
    }
}
```

##### 5. 연산자 우선순위에 대해 설명하고 그에 기반해 아래 코드의 결과를 설명해주세요.(선민)
```java
public class Main {
    public static void main(String[] args) {
        int a = 10;
        int b = 4;

        System.out.println(++a << 1 + b++ * 2 << 1);
    }
}
```
