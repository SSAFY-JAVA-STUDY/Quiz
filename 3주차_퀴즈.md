### 9. 아래 코드의 실행 결과를 설명하세요.(민아)
```java
public class Main {
    public static void main(String[] args) {
        int x = 2;
        int n = 5;
        long result = 0;
        for(int i=1; i<=n; i++){
            result += power(x,i);
        }
        System.out.println(result);
    }
    static long power(int x, int n){
        if(n==1) return x;
        return x * power(x,n-1);
    }
}
```
